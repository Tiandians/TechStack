# 分部积分法

分部积分法的理论基础是**积法则**

-   推导分部积分公式
-   书上归纳了分部积分的四种模式，请说明这些模式的道理
    -   多项式与三角函数、自然指数
    -   多项式与反三角函数、自然对数
    -   三角函数与自然对数，根式
    -   函数的幂

> 快速练习
>
> -   $\int x^2e^x\mathrm{d}x$
